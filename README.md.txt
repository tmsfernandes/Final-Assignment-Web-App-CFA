# Replicant Registry Terminal

The **Replicant Registry Terminal** is a single-page web application designed to simulate a futuristic LAPD system for managing and interacting with replicants. Inspired by the cyberpunk aesthetics of the Blade Runner universe, this project offers functionalities to view, add, delete, and interact with replicants in a retro-themed terminal interface.

---

## Languages and Technologies

### **Frontend**
- **HTML5**: For structuring the web interface.
- **CSS3**: For creating a retro-futuristic terminal-inspired design.
- **JavaScript**: For client-side interactivity and DOM manipulation.

### **Backend**
- **Java (Spring Boot)**: To create the RESTful API.
- **Maven**: For project management and dependencies.
- **In-Memory Data Storage**: For managing replicant data.
